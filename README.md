## Použité technologie
### Client
Vytvořen v Reactu + Vite
- shadcn/ui (komponenty)
- Lucide react (ikony)
- Tailwind CSS (stylování)
- Vlastní CSS
### Server
- MongoDB
- Express
- Mongoose
